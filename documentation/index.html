VTube v2 - Video Hosting & Sharing Script

VTube V2 - Video Hosting & Sharing Script is a script that allows to upload and share videos. Users can upload videos and administrator can approve/reject videos submitted by users. Everyone can watch videos without login but must login to upload/import videos or to comment on videos.

Requirements

Make sure you have:
  • A web server which you can access via FTP or SFTP.
  • A text editor.
  • A web browser.
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
And make sure your web server is running:
  • Web serving software such as Apache. PHP 5.6.3 or later.
  • MySQL 5.x for best performance.
If you are not sure about this, please check with your web hosting provider.

Installation

Easy (Web) Installation -

1 - Create a MySQL database, and a MySQL User with full permissions for that database.

2 - Note down the MySQL details: username, password, database name and server host name. If MySQL is running on the same server as your website, the server host name is likely to be 127.0.0.1 or localhost.

3 - Place all the files from upload_files folder (including .htaccess) to the appropriate location on your web server

4 - Set your domain or subdomain's DOCUMENT ROOT to /<path to public_html>/public

5 - Open the appropriate web page in your web browser, for example

  • http://mysite.com/install

6 - Follow the webinstaller instructions.

if installer doesn't work follow the manual installation.

Manual Installation -

1 - Create a MySQL database, and a MySQL User with full permissions for that database.

2 - Note down the MySQL details: username, password, database name and server host name. If MySQL is running on the same server as your website, the server host name is likely to be 127.0.0.1 or localhost.

3 - Open .env in your text editor, set the following details, and save the file. Do not use a word processor such as Microsoft Word for this, but rather Notepad or another appropriate text editing program.

		
    
		
  1. 
			APP_URL=MY_URL
    2. 
		
  2. 
			DB_HOST=MY_DBHOST
    3. 
		
  3. 
			DB_DATABASE=MY_DBNAME
    4. 
		
  4. 
			DB_USERNAME=MY_DBUSERNAME
    5. 
		
  5. 
			DB_PASSWORD=MY_DBPASSWORD
    6.

4 - Set your domain or subdomain's DOCUMENT ROOT to /<path to public_html>/public

5 - Open the appropriate web page in your web browser, for example:

  • If you installed the script at root folder of a domain, http://www.mysite.com/

6 - To set up your administrator account, navigate  http://www.mysite.com/admin/login
